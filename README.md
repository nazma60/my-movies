# my-movies

A simple and user-friendly React application that allows you to browse trending movies and save your favorites for later.
The app pulls data from a movie API, displays the trending titles, and lets you mark films as favorites using local storage.

**Features**

- Trending Movies

Fetches trending movies from a movie API (TMDB API).

Displays movie posters, titles, and other details in a clean UI.

- Search Feature

Includes a built-in search bar that allows you to look up movies by title.

- Save Your Favorites

Mark any movie as a favorite with a single click.

Favorites are saved locally so they remain available even after refreshing the page.

- Auto-Updating List

Trending movies update automatically based on API data.

**Run command**

npm dev run
